# 실시간 무전기 앱 사용자 가이드

## 개요

이 무전기 앱은 현장 직원들이 스마트폰에서 실시간으로 무전 통신을 할 수 있는 웹 애플리케이션입니다. 1:1 개별 무전과 그룹 무전을 모두 지원하며, 초저지연 실시간 오디오 스트리밍을 제공합니다.

---

## 주요 기능

### 1. 사전 등록 로그인
- **로그인 방식**: 사번과 인증번호를 입력하여 로그인
- **테스트 계정**:
  - 사번: `EMP001` / 인증번호: `1234` (김철수)
  - 사번: `EMP002` / 인증번호: `5678` (이영희)
  - 사번: `EMP003` / 인증번호: `9012` (박민준)
- 로그인 정보는 로컬 저장소에 저장되어 앱을 종료했다가 다시 열어도 유지됩니다.

### 2. 상시 활성화 대기 상태
- 앱 상단에 **초록색 깜빡이는 인디케이터**와 **"무전 수신 대기 중..."** 텍스트가 항상 표시됩니다.
- 이는 앱이 언제든 무전을 수신할 준비가 되어 있음을 나타냅니다.

### 3. 1:1 개별 무전
- **동료 목록**: 화면 좌측에 현재 로그인한 사용자를 제외한 모든 동료가 표시됩니다.
- **상대 선택**: 동료 이름을 클릭하여 무전할 상대를 선택합니다.
- **거대한 무전 버튼**: 상대를 선택하면 화면 중앙에 큰 무전 버튼이 나타납니다.
- **토글 작동**:
  - **대기 상태(회색)**: 버튼을 누르면 송신 상태로 전환
  - **송신 상태(초록)**: 마이크를 통해 음성을 전송 중. 다시 누르면 송신 종료
  - **수신 상태(파란색)**: 상대방의 무전을 수신 중. 송신 버튼은 비활성화됨

### 4. 그룹 무전
- **그룹 무전 전환 버튼**: 화면 하단의 "그룹 무전 전환" 버튼을 클릭하여 그룹 무전 모드로 전환
- **Bottom Sheet**: 그룹 무전 모드는 화면 하단에서 위로 올라오는 시트 형태로 표시됩니다.
- **그룹 선택**: 사용 가능한 그룹 목록에서 원하는 그룹을 선택합니다.
- **그룹 무전**: 1:1 무전과 동일한 방식으로 토글 버튼을 사용하여 그룹 무전을 시작/종료합니다.
- **1:1 모드로 복귀**: Bottom Sheet를 닫으면 다시 1:1 무전 모드로 돌아갑니다.

## 버튼 상태 및 색상

| 상태 | 색상 | 의미 |
|------|------|------|
| **대기** | 회색 | 무전 준비 완료, 송신 가능 |
| **송신** | 초록 | 현재 음성 송신 중 |
| **수신** | 파란색 | 상대방의 음성 수신 중, 송신 불가 |

## 사용 방법 (단계별)

### 1단계: 로그인
1. 앱을 열면 로그인 화면이 표시됩니다.
2. 사번과 인증번호를 입력합니다.
3. "로그인" 버튼을 클릭합니다.

### 2단계: 1:1 무전 시작
1. 좌측 동료 목록에서 무전할 상대를 선택합니다.
2. 상대 이름이 강조되고, 화면 중앙에 거대한 무전 버튼이 나타납니다.
3. 무전 버튼을 누르면 송신이 시작됩니다. (버튼이 초록색으로 변함)
4. 말을 마친 후 다시 버튼을 누르면 송신이 종료됩니다.

### 3단계: 그룹 무전 시작
1. 화면 하단의 "그룹 무전 전환" 버튼을 클릭합니다.
2. Bottom Sheet가 올라오고 그룹 목록이 표시됩니다.
3. 원하는 그룹을 선택합니다.
4. 1:1 무전과 동일한 방식으로 무전 버튼을 사용합니다.

### 4단계: 로그아웃
1. 화면 우측 상단의 "로그아웃" 버튼을 클릭합니다.
2. 로그인 화면으로 돌아갑니다.

## 기술 사양

### 오디오 스트리밍
- **Web Audio API**: 마이크 입력 캡처 및 오디오 처리
- **실시간 전송**: 오디오 데이터를 청크 단위로 즉시 전송
- **메모리 효율**: 수신한 오디오는 재생 후 즉시 파기되며, 서버에 저장되지 않음

### 데이터베이스
- **사용자 테이블**: 60명의 사전 등록된 사용자 관리
- **그룹 테이블**: 팀별 그룹 정의
- **그룹 멤버 테이블**: 각 그룹에 속한 사용자 관리

### 보안
- **사전 등록 인증**: 승인된 사용자만 로그인 가능
- **로컬 저장소**: 로그인 정보는 브라우저 로컬 저장소에만 저장
- **오디오 보안**: 오디오 데이터는 실시간 전송되며 서버에 저장되지 않음
